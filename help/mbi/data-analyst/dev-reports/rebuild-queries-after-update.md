# 업데이트 주기 후 쿼리 다시 작성

쿼리를 다시 빌드할 필요는 없습니다. 를 사용하여 생성된 보고서 [`SQL Report Builder`](../dev-reports/sql-rpt-bldr.md) 기존 템플릿에서 생성된 템플릿처럼 저장됩니다 `Report Builder`. SQL 차트에 대한 갱신 프로세스는 동일합니다. 데이터가 갱신되면 차트의 값이 다시 계산되고 다시 표시됩니다.
