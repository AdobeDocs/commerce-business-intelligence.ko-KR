# 업데이트 주기 후 질의 재구축

쿼리를 다시 빌드할 필요는 없습니다. 보고서를 사용하여 만든 보고서 [`SQL Report Builder`](../dev-reports/sql-rpt-bldr.md) 기존 제품에서 생성된 와 마찬가지로 저장됩니다. `Report Builder`. SQL 차트의 업데이트 프로세스는 동일합니다. 데이터가 업데이트되면 차트의 값이 다시 계산되고 다시 표시됩니다.
