# [!DNL PrestaShop] 연결

![PrestaShop 로고](../../../assets/Prestashop-logo.png)

비즈니스의 성과를 확인하거나 가장 중요한 확보 소스 및 채널을 찾고 싶으십니까? [!DNL PrestaShop] 데이터를 [!DNL Commerce Intelligence]에 연결하면 이러한 질문에 답변할 수 있습니다.

[!DNL PrestaShop] 데이터베이스는 [!DNL MySQL]을(를) 기반으로 합니다. `SSH tunnel`[&#128279;](../integrations/mysql-via-ssh-tunnel.md)을(를) 통해 [직접 연결](../integrations/mysql-via-a-direct-connection.md), [[!DNL cPanel]](../integrations/mysql-via-cpanel.md) 또는 을(를) 통해 [!DNL Commerce Intelligence]에 연결할 수 있습니다. Adobe에서는 가장 안전하므로 `SSH` 메서드를 사용하는 것이 좋습니다. SSH 호스트 키 확인에 대해서는 [SSH 호스트 키 확인](../integrations/ssh-host-key-verification.md)을 참조하십시오.
