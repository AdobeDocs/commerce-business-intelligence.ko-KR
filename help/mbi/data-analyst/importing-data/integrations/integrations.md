# 통합을 사용하여 외부 데이터 연결

통합은 데이터 소스입니다. 통합은 데이터 웨어하우스에 연결할 수 있는 응용 프로그램 및 데이터베이스입니다.

이 섹션에서는 다양한 외부 데이터 소스를 [!DNL MBI].

연결 또는 통합을 삭제하려면 [지원](../../../guide-overview.md) 삭제할 연결 또는 통합의 이름을 포함합니다.
